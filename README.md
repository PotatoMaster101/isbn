# ISBN Checker
Quick and dirty ISBN checker in [.Net core](https://dotnet.microsoft.com).

## Usage
```
$ isbn [isbn1] [isbn2] ...
```

## Building
Requires [`dotnet`](https://dotnet.microsoft.com/download) to build.
```
$ git clone https://github.com/PotatoMaster101/isbn
$ cd isbn
$ dotnet build -c Release
```

## Testing
```
$ dotnet test
```
